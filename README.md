# HQ_Help
*接口已加固，本脚本方法失效，不再更新*

冲顶大会辅助答题工具,帮助你更好答题（可能也会帮倒忙，哈哈哈）
此工具是从接口获取题目及选项，然后采取三种方式检索答案
1. 直接打开浏览器，搜索引擎搜索，例如百度
2. 题目+选项搜索结果计数法
3. 题目搜索结果包含选项词频计数法
具体哪种好请各位自己判别
